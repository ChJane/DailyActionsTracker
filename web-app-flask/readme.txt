Инструкция по запуску приложения

1. Активируйтевиртуальное окружение. 
Для этого в командной строке перейдите в данную папку и наберите:

venv\Scripts\activate

2. Установим нужные пакеты. Для этого наберите:

pip install -r req.txt

Если ваш python установлен в папку C:\Program Files\..., то cmd должен быть запущен с правами администратора.

3. Если на предыдущем шаге все пакеты установились, то наберите:

set FLASK_APP=project.py
flask run

Вы должны увидеть такую запись:
 * Serving Flask app "project"
 * Running on http://127.0.0.1:5000/ (Press CTRL+C to quit)

Если это так и есть, то переходите по адресу http://127.0.0.1:5000/
